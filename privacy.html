SocialAvatar 隱私權政策 Privacy Policy

最後更新 / Last updated: 2025-10-17

SocialAvatar（以下稱「本服務」）重視您的個人資料與隱私。我們依據本政策說明如何在您同意授權後， 透過 Instagram Graph API 取得並處理資料，以生成您的動態人物圖與社群人格分析。

1. 我們蒐集哪些資料 / What we collect

  • 您透過 Instagram Graph API 授權提供的資料，例如：
    IG 使用者名稱、頭像網址、自我介紹（biography）、粉絲數（followers_count）、 追蹤數（follows_count）、貼文數（media_count）、貼文內容（caption）及貼文媒體連結（media_url）。
  • 必要的服務資料：登入時間、裝置/瀏覽器資訊、推薦碼/短連結參數。
  • 付款或點數交易（如有）：僅保存交易必要資訊（由金流服務商處理信用卡資料）。

2. 我們如何使用 / How we use data

  • 產生動態人物圖與「社群人格」分析（娛樂用途）。
  • 計算粉絲等級、載具外觀與限量掉落資格。
  • 記錄推薦轉換、發放點數、偵測濫用。
  • 改進產品體驗、統計與除錯（以彙總/匿名方式）。

3. 資料保存與刪除 / Retention & Deletion

我們僅在提供服務所需期間保存您的資料；您可隨時依下方「資料刪除」流程要求刪除。我們也可能依法令或爭議處理需求，保留最短必要期間之紀錄。

4. 第三方分享 / Sharing

我們不會販售您的個人資料。必要時可能與以下對象分享：雲端儲存/托管供應商、金流服務商、分析服務（僅限必要最小化資料）。 所有處理均依本政策與合約。

5. 安全 / Security

採用傳輸/靜態加密、權限控管與存取稽核等措施；然而網路服務無法保證絕對安全。

6. 您的權利 / Your rights

您可要求查閱、更正、下載或刪除您的資料，並可隨時撤回授權。請見下方聯絡方式。

7. 與 Meta/Instagram 的關係 / Relation to Meta

SocialAvatar 與 Meta/Instagram 無關，也未獲其背書或贊助。本服務之資料處理僅在您授權且遵守 Meta 平台政策的前提下進行。
